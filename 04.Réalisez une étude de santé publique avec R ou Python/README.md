# Projet 4 – Réalisez une étude de santé publique  
## Analyse statistique et exploration de données avec Python

## Description du projet  
L’objectif de ce projet était d’analyser un jeu de données de santé publique afin d’identifier les facteurs influençant l’état de santé d’une population.  
L’étude a consisté à nettoyer les données, explorer les relations entre les variables, construire des visualisations et réaliser des analyses statistiques pertinentes pour répondre à une problématique de santé publique.

## Objectifs  
- Nettoyer et préparer les données pour l’analyse.  
- Étudier la distribution des variables et leurs corrélations.  
- Identifier les facteurs ayant un impact significatif sur un indicateur de santé.  
- Réaliser des analyses statistiques adaptées (tests, corrélations, régressions selon le cas).  
- Produire des visualisations claires facilitant l’interprétation.  
- Présenter des conclusions exploitables pour la prise de décision en santé publique.

## Compétences développées  
- Préparation et nettoyage des données (pandas, numpy).  
- Analyse exploratoire des données.  
- Visualisation avec matplotlib et seaborn.  
- Analyses statistiques.  
- Interprétation de résultats en contexte santé publique.  
- Structuration d’un rapport d’analyse clair et cohérent.

## Outils utilisés  
### Python  
- Pandas : nettoyage et manipulation des données  
- NumPy : statistiques descriptives  
- Matplotlib et Seaborn : visualisations  

## Résultats  
- Mise en évidence des tendances et disparités dans la population étudiée.  
- Identification des facteurs les plus associés à l’indicateur de santé analysé.  
- Visualisations permettant de mieux comprendre les relations entre variables.  
- Synthèse des observations pertinentes pour les acteurs de santé publique.

## Conclusion  
Ce projet m’a permis de mobiliser mes compétences en analyse de données et en statistiques pour répondre à une problématique de santé publique.  
Il illustre ma capacité à transformer des données brutes en informations utiles pour éclairer une prise de décision.
