## Description du projet  
Ce projet consiste à concevoir un tableau de bord interactif avec Power BI afin de visualiser l’avancement de plusieurs projets. L’objectif était de transformer un ensemble de données brutes en indicateurs clés et visualisations permettant de suivre efficacement la progression, d’identifier les retards et de faciliter la prise de décision.

## Objectifs  
- Importer, nettoyer et préparer les données dans Power BI.  
- Mettre en place un modèle de données adapté au suivi de projets.  
- Créer des mesures DAX pour calculer les indicateurs nécessaires (avancement, retards).  
- Concevoir un tableau de bord dynamique permettant de filtrer par projet, équipe, période, ou état d’avancement.  
- Produire une visualisation claire, lisible et adaptée aux besoins métiers.

## Compétences développées  
- Nettoyage et transformation des données avec Power Query.  
- Modélisation dans Power BI : relations entre tables, schéma en étoile, optimisation du modèle.  
- Création de mesures DAX pour l’analyse d’indicateurs métier.  
- Conception de tableaux de bord interactifs : filtres, slicers, navigation.  
- Visualisation de données : choix de graphiques pertinents, hiérarchisation des informations.  
- Communication analytique : restitution claire des insights et des performances des projets.

## Outils utilisés  
- Power BI Desktop  
- Power Query pour la préparation des données  
- DAX pour les calculs d’indicateurs  
- Modèle de données Power BI  
- Tableaux de bord interactifs et visualisations personnalisées

## Résultats  
- Tableau de bord complet permettant le suivi de plusieurs projets.  
- Indicateurs clés calculés : taux d’avancement, nombre de tâches en retard, charge restante, prévisions.  
- Visualisations interactives : courbes d’évolution, diagrammes d’état, cartes de suivi par équipe ou par tâche.  
- Interface ergonomique permettant d’identifier rapidement les points critiques.  
- Un outil réutilisable et adaptable pour d'autres contextes de gestion de projets.

## Conclusion  
Ce projet m’a permis de consolider mes compétences en visualisation de données et en modélisation Power BI. J’ai appris à concevoir un tableau de bord complet et dynamique, capable de synthétiser des données complexes pour offrir une vue claire de l’état d’avancement des projets.
