# Projet 6 – Optimisez la gestion des données d’une boutique avec R ou Python

## Description du projet  
Dans ce projet, j’ai travaillé sur l’optimisation de la gestion des données d’une boutique. L’objectif était de nettoyer, transformer et analyser un ensemble de données afin d’améliorer la qualité des informations disponibles pour la prise de décision. 

## Objectifs  
- Nettoyer et préparer un jeu de données contenant les informations d’une boutique.  
- Détecter les incohérences : doublons, données manquantes, formats incorrects.  
- Structurer les données pour faciliter l’analyse (normalisation, typage, gestion des dates).  
- Réaliser des analyses descriptives pour mieux comprendre les ventes, les produits et les clients.  
- Produire des visualisations permettant de dégager des tendances utiles.  
- Générer un rapport d’analyse clair et exploitable.

## Compétences développées  
- Nettoyage de données : traitement des valeurs manquantes, détection des anomalies, correction des formats.  
- Manipulation de données : fusion, regroupement, filtrage, transformation.  
- Analyse statistique descriptive : tendances générales, distribution des ventes, segmentation produits/clients.  
- Visualisation de données : graphiques pour explorer les performances commerciales.  
- Optimisation et organisation : structuration du code et automatisation de certaines opérations.  
- Documentation : production d’un rapport clair présentant les méthodes et les résultats.

## Outils utilisés  

### Python  
- Python  
- Pandas : manipulation et nettoyage des données  
- NumPy : opérations numériques  
- Matplotlib / Seaborn : visualisation  
- Jupyter Notebook 

## Résultats  
- Jeu de données entièrement nettoyé et structuré.  
- Identification des produits les plus performants et des catégories problématiques.  
- Mise en évidence des créneaux horaires ou périodes avec les meilleures ventes.  
- Amélioration de la qualité des données facilitant les analyses futures de la boutique.  
- Rapport final détaillant les traitements réalisés, les analyses obtenues et les pistes d’amélioration.

## Conclusion  
Ce projet m’a permis de renforcer mes compétences en nettoyage et manipulation de données, ainsi qu’en analyse statistique et visualisation. J’ai appris à transformer un jeu de données brut en un tableau clair, organisé et exploitable, facilitant la prise de décision pour une boutique.
