## Description du projet  
L’objectif de ce projet était d’analyser les ventes d’une librairie afin de comprendre les performances commerciales, d’identifier les tendances et de repérer les axes d’amélioration. Le travail a consisté à nettoyer les données, explorer les indicateurs clés et réaliser des visualisations permettant de synthétiser les résultats.

## Objectifs  
- Nettoyer et préparer les données de ventes de la librairie.  
- Explorer les performances des livres, catégories et auteurs.  
- Identifier les tendances de vente selon les périodes (jour, semaine, mois).  
- Étudier le comportement des clients et leur fidélité.  
- Produire des visualisations permettant d’expliquer clairement les résultats.  
- Rédiger une analyse finale présentant les insights essentiels.

## Compétences développées  
- Nettoyage et manipulation de données avec Python.  
- Analyse des ventes : volumes, répartition par catégories, rentabilité.  
- Visualisation de données pour représenter les tendances et les écarts.  
- Analyse du comportement client (récurrence, panier moyen, segmentation simple).  
- Structuration d’un workflow analytique complet : du traitement à l'interprétation.  
- Communication des résultats à travers un rapport clair.

## Outils utilisés  
### Python  
- Python  
- Pandas : manipulation et nettoyage des données  
- NumPy : opérations statistiques de base  
- Matplotlib / Seaborn : visualisations  
- Jupyter Notebook ou script Python  

## Résultats  
- Évaluation des performances des catégories de livres et identification des best-sellers.  
- Analyse du chiffre d’affaires par période et détection des saisons les plus actives.  
- Visualisation des ventes par genre et par catégorie de livres.  
- Mise en évidence de comportements clients : fréquence d’achat, panier moyen, fidélité.  
- Synthèse claire des leviers potentiels pour améliorer les ventes de la librairie.

## Conclusion  
Ce projet m’a permis d’approfondir mes compétences en analyse de données commerciales, de la phase de nettoyage jusqu’à l’interprétation des résultats. L’utilisation de Python m’a permis d’automatiser certaines étapes et de produire des visualisations pertinentes pour la prise de décision.
