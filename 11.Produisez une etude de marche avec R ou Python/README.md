# Projet 11 – Étude de marché pour l’exportation de poulet  
## Analyse de groupement de pays avec Python

## Description du projet  
L’objectif de ce projet était de réaliser une étude de marché pour identifier les pays les plus intéressants pour l’exportation de poulet. À partir de données économiques, démographiques et commerciales, l’analyse a permis de segmenter les pays en groupes homogènes afin de mieux cibler les marchés potentiels.  
Le projet a été réalisé avec Python.

## Objectifs  
- Explorer et préparer les données relatives aux pays (économie, population, consommation, commerce).  
- Identifier des groupes de pays présentant des caractéristiques similaires pour l’exportation.  
- Effectuer un groupement statistique (clustering) pour segmenter les marchés.  
- Visualiser les segments de manière claire pour faciliter la prise de décision.  
- Produire une synthèse des opportunités et recommandations pour l’exportation.

## Compétences développées  
- Préparation et nettoyage des données avec Python (gestion des valeurs manquantes, typage).  
- Analyse exploratoire des données et compréhension des variables clés.  
- Segmentation de pays par clustering (k-means ou autres méthodes).  
- Visualisation des clusters et interprétation des résultats.  
- Communication analytique des insights pour une stratégie d’exportation.

## Outils utilisés  
### Python  
- Pandas : manipulation et nettoyage des données  
- NumPy : calculs et statistiques  
- Scikit-learn : clustering (k-means, standardisation)  
- Matplotlib / Seaborn : visualisation  
- Jupyter Notebook : documentation et analyse  

## Résultats  
- Segmentation des pays en plusieurs groupes selon leurs caractéristiques économiques et commerciales.  
- Identification des clusters les plus favorables à l’exportation de poulet.  
- Visualisation claire des groupes de pays pour faciliter la prise de décision.  
- Synthèse des marchés cibles et recommandations stratégiques pour l’entreprise.

## Conclusion  
Ce projet m’a permis d’appliquer des techniques d’analyse de données avancées pour une problématique commerciale concrète.  
J’ai acquis de l’expérience dans le clustering, l’interprétation des résultats et la visualisation pour guider des décisions d’exportation.
