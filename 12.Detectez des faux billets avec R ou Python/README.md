# Projet 12 – Détection de faux billets  
## Machine learning et analyse multivariée avec Python

## Description du projet  
L’objectif de ce projet était de concevoir un modèle capable de distinguer de vrais billets de faux billets à partir de leurs caractéristiques physiques.  
Le projet a combiné **analyse en composantes principales (ACP)**, **clustering (K-means)** et modèles supervisés (**régression linéaire multiple** et **Random Forest**) pour explorer, segmenter et classifier les billets.

## Objectifs  
- Explorer et comprendre les variables du jeu de données.  
- Réduire la dimensionnalité avec ACP pour identifier les axes principaux et les corrélations.  
- Segmenter les billets avec K-means pour visualiser les groupes naturels.  
- Construire des modèles de classification : régression linéaire multiple et Random Forest.  
- Évaluer les performances des modèles et identifier le modèle le plus efficace.  
- Interpréter les résultats pour comprendre les caractéristiques discriminantes.

## Compétences développées  
- Nettoyage et préparation des données avec Python (pandas, numpy).  
- Analyse exploratoire et visualisation des données (matplotlib, seaborn).  
- Réduction de dimensionnalité avec ACP.  
- Clustering avec K-means pour identifier des groupes de billets similaires.  
- Modélisation supervisée : régression linéaire multiple et Random Forest.  
- Évaluation des modèles : précision, recall, matrice de confusion.  
- Interprétation des résultats et des caractéristiques les plus discriminantes.

## Outils utilisés  
### Python  
- Pandas : manipulation et transformation des données  
- NumPy : calculs et statistiques  
- Matplotlib et Seaborn : visualisations  
- Scikit-learn :  
  - ACP (PCA)  
  - K-means  
  - Régression linéaire multiple  
  - Random Forest  
  - Évaluation des modèles  

## Résultats  
- Identification des caractéristiques les plus importantes grâce à ACP.  
- Segmentation des billets par K-means pour visualiser des patterns naturels.  
- Modèles de classification entraînés : régression linéaire multiple et Random Forest.  
- Random Forest retenu comme modèle le plus performant pour la détection de faux billets.  
- Visualisation des résultats et des erreurs du modèle pour comprendre ses limites.
